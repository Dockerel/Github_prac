## **현재 디렉터리 살펴보기**

* pwd : 현재 위치의 경로

* ls : 현재 디렉터리에 있는 파일 혹은 디렉터리
   - **ls -la** or **ls -al** : 숨긴 파일과 디렉터리 표시
   
      * -a : 숨긴 파일과 디렉터리도 함께 표시
      * -l : 파일이나 디렉터리의 상세 정보를 함께 표시
      * -r : 파일의 정렬 순서를 거꾸로 표시
      * -t : 파일 작성 시간 순으로 (내림차순) 표시
